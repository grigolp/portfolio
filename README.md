# Portfolio Website

Personal portfolio website showcasing ML engineering projects, AWS expertise, and professional experience.

## Quick Start

1. Clone the repository
2. Open `index.html` in a browser

## Deployment

Deployed on GitHub Pages at: `https://grigolp.github.io/portfolio/`

### Update Deployment

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

Changes will be live in 1-2 minutes.

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript
- GitHub Pages

## Project Structure

```
portfolio/
├── index.html              # Main landing page
├── project-*.html          # Individual project pages
├── styles.css              # All styling
├── script.js               # Navigation & interactions
└── assets/
    └── images/             # Logos & project images
```
