The Challenge

Kasparium developed an innovative warehouse where storage racks move to workers, rather than workers walking to racks. The core challenge: coordinate parallel rack movements without collisions.

Technical Complexity

  • Multi-agent pathfinding: 10+ orders moving simultaneously
  • Priority management: High-priority orders must complete first
  • Limited empty spaces: Efficient use of floor space required
  • Station traffic control: Limit concurrent deliveries per station
  • Deadlock prevention: Circular dependencies must be avoided
  • Real-time performance: Sub-second response for next-move decisions

Technical Solution

IDA* Pathfinding with Custom Heuristics

Implemented Iterative Deepening A* for memory-efficient pathfinding with custom cost functions:

# Multi-component cost heuristic def calculate_cost(order, station, empties): # Primary: Distance to target station cost_target = order.distance(station) # Secondary: Proximity to empty tiles (enables future moves) cost_empty = avg([order.distance(e) for e in empties]) / 5 # Tertiary: Position empties near stations for return paths cost_empty_station = avg([station.distance(e) for e in empties]) / 10 return cost_target + cost_empty + cost_empty_station
