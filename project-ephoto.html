The Challenge

Official document photos have strict ICAO (International Civil Aviation Organization) compliance requirements. Approximately 55% of non-expert or DIY photos fail due to:

  • Incorrect head positioning and pose
  • Lighting and shadow issues
  • Facial expression violations (smiling, eyes closed, mouth open)
  • Distance and framing problems
  • Attire and accessory violations

Why This Matters: Photo rejections cause application delays, missed deadlines, and significant user anxiety for time-sensitive processes like DV Lottery applications and visa renewals.

Technical Constraints

  • ✓ Cost Efficiency: Process high volume at <$0.10/image
  • ✓ Accuracy: Zero tolerance for false positives
  • ✓ Edge Cases: Handle diverse user scenarios
  • ✓ Real-time Feedback: Quick validation results
  • ✓ Regulatory Compliance: Encode complex ICAO specifications

Technical Solution

Two-Stage Validation Architecture

The system orchestrates AWS Rekognition, SageMaker, and OpenAI GPT-4 Vision to achieve automated ICAO compliance checking with human-level accuracy.

Key Design Decisions

1. Two-Stage Validation Architecture

Rationale: Rekognition costs ~$0.001/image vs GPT-4 Vision at ~$0.04/image

Strategy: Use cheap automated checks to filter out obvious failures before expensive AI validation

Impact: 60% cost reduction while maintaining accuracy

2. Payment-Gated AI Validation

Rationale: Only paying customers receive expensive GPT-4 Vision validation

Strategy: Stage 1 provides free pre-check; Stage 2 activates post-payment

Impact: Protects against abuse while maintaining 23% conversion rate

3. SageMaker for Preprocessing

Rationale: Need consistent, scalable image preprocessing

Strategy: Centralized preprocessing handles background removal, normalization, basic checks

Impact: Reduced Elastic Beanstalk compute load, improved response times

GPT-4 Vision Prompt Engineering

The system implements structured prompt engineering to ensure GPT-4 acts as a deterministic compliance validator:

# Prompt engineering for GPT-4 Vision SAMPLED_COMPLIANCE_PROMPT = """ You are an ICAO photo compliance validator. Analyze this official document photo. REQUIRED CHECKS: 1. Facial Expression: Must be neutral (no smiling, serious expression) 2. Eyes: Both fully open and visible 3. Mouth: Closed (teeth not visible) 4. Head Position: Centered, straight, facing camera directly 5. Lighting: Even, no harsh shadows on face 6. Attire: Appropriate (no uniforms, costumes, or casual wear) 7. Accessories: No sunglasses, hats, or visible headphones 8. Background: Uniform, neutral color 9. Image Quality: Sharp focus, no blur or pixelation 10. Framing: Proper head-to-shoulder ratio OUTPUT FORMAT (JSON): { "is_compliant": true/false, "violations": ["list of specific violations"], "confidence": 0-100, "reasoning": "brief explanation" } Be strict. When in doubt, reject. """
